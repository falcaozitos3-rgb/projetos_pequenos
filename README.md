# Projeto: Site de Identificação (Flask + HTML/CSS)

Este é um projeto prático desenvolvido para treinar a criação e integração de sites utilizando **Python** no backend e **HTML/CSS** com frontend. 

##  Sobre o Projeto
A ideia surgiu como um desafio pessoal para aplicar conhecimentos de lógica e web design. Após conversar com meu professor, decidi desenvolver esta aplicação para apresentar em sala de aula e demonstrar o funcionamento de uma aplicação web real.

##  Tecnologias Utilizadas
*   **Python**: Linguagem principal.
*   **Flask**: Micro-framework para o servidor web.
*   **HTML/CSS**: Para a estrutura e o design visual do site.

##  Como rodar o projeto
1. Instale o Flask: `pip install flask`
2. Execute o arquivo principal: `python app.py`
3. Acesse no navegador: `http://127.0.0.1:5000`

---
*Projeto desenvolvido por mim para fins de estudo e apresentação acadêmica.*
